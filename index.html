# 勤務実績表自動作成システム 給与計算をもっと簡単に、正確に、自動化するWebアプリケーションです。 ## 🏆 主な特徴 ### ✨ 究極の効率化を実現 - **設定自動引き継ぎ**: 一度設定すれば翌月も自動継続 - **過去データ統合管理**: 既存ファイルに新シートとして追加保存 - **リアルタイム自動計算**: 残業代・給与計算が瞬時に完了 ## 📋 現在実装済みの機能 ### 💼 従業員基本設定 - 氏名・対象月・雇用形態（常勤・パート）の管理 - **常勤**: 月給制 + 残業代計算 - **パート**: 時給制 + 残業代計算 - 通勤手当・調整手当の複数設定対応 - 有給休暇の管理 ### 📅 曜日別詳細設定 - 月曜日〜日曜日の個別設定 - 残業開始時間・休憩時間・残業単価の設定 - 一括設定機能も提供 ### 📊 データ処理・計算機能 - タイムカードCSV/Excelファイルのインポート - 期間自動生成（月単位） - 休日出勤対応 - **課税対象・非課税支給額**の自動分類 - **詳細な給与計算式**の出力 ### 🔄 設定管理 - **JSON形式**での設定エクスポート・インポート - **過去の勤務実績表**からの設定復元 - 設定の使い回し・バックアップ機能 ### 📤 多形式ファイル出力 - **Excel (.xlsx)** 形式でのエクスポート - **CSV** 形式でのエクスポート - **🏆 既存ファイルへの追加保存**: 過去データも1ファイルで統合管理 ### 🎯 高精度自動計算 - 実働時間・残業時間の自動計算 - **課税対象支給額** と **非課税支給額** の分離 - **総支給額** まで詳細に自動計算 - **計算式の詳細表示** ## 🌐 公開URL - **本体アプリ**: [index.html](./) - **機能説明**: [help.html](./help.html) ## 🔧 技術仕様 ### データモデル・構造 ```javascript // 従業員基本設定 employeeSettings: { name: string, // 従業員名 targetMonth: string, // 対象年月 (YYYY-MM) employmentType: string, // 雇用形態 (fulltime/parttime) totalPaidLeaveDays: number, usedPaidLeaveDays: number, wageSettings: { // パートタイム regularHourlyWage: number, overtimeHourlyWage: number, commutingAllowance: number, // 常勤 baseSalary: number, // 共通 adjustmentAllowances: Array<{name: string, amount: number}> } } // 曜日別設定 daySettings: { [day: string]: { overtimeStart: string, // HH:MM形式 breakTime: number, // 分 overtimeRate: number // 円/時 } } // 勤務データ workData: Array<{ date: string, // YYYY-MM-DD startTime: string, // HH:MM endTime: string, // HH:MM breakDuration: string, // 休憩時間 isHolidayWork: boolean // 休日出勤フラグ }> ``` ### ストレージ - **ローカルストレージ**: ブラウザ内でデータ保持 - **ファイルエクスポート**: JSON、Excel、CSV形式 - **データ統合**: 既存Excelファイルへのシート追加 ## 🚀 未実装機能・今後の開発予定 ### 📈 機能拡張 - [ ] 複数従業員の一括管理 - [ ] 年次有給休暇の自動計算 - [ ] 社会保険料・税金計算 - [ ] 勤怠統計・レポート機能 ### 🔧 UI/UX改善 - [ ] ダークモード対応 - [ ] 印刷レイアウトの最適化 - [ ] モバイル表示の更なる改善 ### 🔗 連携機能 - [ ] 他社勤怠システムとの連携 - [ ] クラウドストレージ対応 - [ ] API連携機能 ## 💡 推奨される次のステップ 1. **複数従業員管理**: 部署・チーム単位での一括処理 2. **テンプレート機能**: 業種別の設定テンプレート 3. **承認ワークフロー**: 上司承認機能の実装 4. **データ分析**: 労働時間分析・コスト分析機能 ## 🎯 プロジェクトの目標 **給与計算業務の完全自動化**を目指し、以下を実現： - ⚡ **作業時間80%短縮**: 設定の自動引き継ぎにより - 🎯 **計算ミスゼロ**: 自動計算による人的エラー排除 - 📊 **データ統合管理**: 1ファイルですべての期間を管理 - 🔄 **業務フロー改善**: 既存業務への自然な組み込み --- ## ⚡ クイックスタート 1. **基本設定**: 従業員情報と給与設定を入力 2. **曜日別設定**: 勤務条件を設定（一括設定も可能） 3. **データ入力**: タイムカードをインポートまたは手動入力 4. **エクスポート**: Excel・CSV形式で出力 **2ヶ月目以降は設定が自動引き継がれ、データ入力だけで完了！** 詳しい使い方は [機能説明ページ](./help.html) をご覧ください。
